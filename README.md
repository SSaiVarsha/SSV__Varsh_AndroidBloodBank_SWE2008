The proposed system provides distinct modules for donors and receivers:

Receiver Features

Find Nearby Donors: Search by blood type and location.

Find Blood Banks: Contact blood banks via JustDial integration.

Find Hospitals: Access hospital directories and contact options.

Notifications: View updates about requests or availability.

Profile Management: Edit details and manage sign-out securely.

Donor Features

View Available Requests: Respond to nearby blood requests.

Find Donation Centers or Hospitals: Navigate using integrated JustDial search.

Track My Donations: View donation history and update status.

Profile Management: Edit personal information and sign out.

The app ensures smooth navigation through a modern Android interface, using Intents for activity transitions and XML layouts for a clean UI.
